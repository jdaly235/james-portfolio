Essential storage

A small browser-storage value records whether you selected essential-only or allowed preferences. This prevents the banner from appearing on every visit.

When allowed, the site remembers whether you selected the light or dark colour theme. If you choose essential-only, theme changes apply only for the current page session.

When you submit the form, the details you enter are sent to a private Discord channel through the contact API. The API stores an enquiry reference, timestamp, submitted fields and a one-way hash of the source IP for rate limiting and audit purposes.
